Research Intelligence

Welcome

Research Intelligence monitors academic papers and GitHub projects, scores them with AI, and learns your preferences over time. This setup will configure your instance.

You'll configure:
• API key for AI scoring
• Which research domains to monitor
• GitHub & events tracking
• Pipeline schedule

API Key

An Anthropic API key is required for paper scoring. It will be stored in a local .env file, not in the config.

Get one at console.anthropic.com
